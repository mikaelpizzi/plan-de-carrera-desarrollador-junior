# Hoja de Ruta del Desarrollador Junior: De Cero a Contratado

Hola, soy Mikael. Este repositorio no es un proyecto de código, es el "mapa de conocimientos" que he desarrollado basándome en mi experiencia como ingeniero senior y mentor.

Muchos graduados de bootcamp están atascados enviando cientos de CVs sin respuesta. Esto no es un fallo de habilidad, es un fallo de estrategia.

Este mapa detalla el sistema exacto que enseño para cerrar la brecha entre la teoría del bootcamp y las demandas del mercado laboral real.

---

## El Sistema: Los 6 Pilares Estratégicos

El éxito en la contratación no depende de un solo activo, sino de un ecosistema profesional optimizado.

### Pilar 1: El Currículum (CV)
* **El Error:** Tu CV está diseñado para humanos (2 columnas, gráficos), pero es rechazado por máquinas (ATS).
* **La Estrategia:** Debes optimizar para una doble audiencia: primero la máquina (formato de 1 columna, PDF de texto), y segundo el humano (cuantificando logros con la fórmula **Acción + Problema = Resultado**).

### Pilar 2: El Portafolio Web
* **El Error:** Tu portafolio es una "galería" de clones de tutoriales (clon de Netflix, app del clima).
* **La Estrategia:** Tu portafolio debe ser tu "Proyecto Cero", un producto en sí mismo. Debes presentar 3-4 "Estudios de Caso" que usen la narrativa STAR (Problema -> Proceso -> Solución -> Resultado) para explicar el *porqué* de tus decisiones.

### Pilar 3: El Perfil de GitHub
* **El Error:** Tu GitHub es un "cementerio" de proyectos sin `README.md` y commits genéricos ("fix").
* **La Estrategia:** Tu perfil debe ser curado. El `Profile README` es tu centro de mando. Los proyectos anclados deben tener `README.md` detallados, pruebas (tests) y commits semánticos.

### Pilar 4: El Perfil de LinkedIn
* **El Error:** Tu perfil es un CV pasivo con un titular genérico ("Junior Developer").
* **La Estrategia:** Tu perfil debe ser una página de ventas. Optimiza tu titular para el stack (ej. "Desarrollador React | TypeScript") y aprueba las "Evaluaciones de Aptitudes" (Badges) para ganar visibilidad algorítmica.

### Pilar 5: Habilidades Técnicas (La Brecha Oculta)
* **El Error:** Estás atrapado en la "burbuja de tutoriales" de MERN (sobre todo stacks sin bases de datos relacionales).
* **La Estrategia:** La mayoría de los trabajos corporativos demandan bases de datos relacionales (SQL). Debes demostrar competencia en SQL (ej. PostgreSQL) para diferenciarte del resto de graduados de bootcamp.

### Pilar 6: Habilidades Blandas
* **El Error:** Listar "Comunicación" o "Trabajo en Equipo" en tu CV.
* **La Estrategia:** Debes *demostrar* estas habilidades. La "Coachability" (capacidad de ser entrenado) y el "Arte de Pedir Ayuda Eficazmente" son las habilidades blandas más críticas que busca un gerente al contratar a un junior.
